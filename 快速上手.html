📚 5 分鐘上手指南

從零打造一份有圖、有故事、有題目的 Google 表單測驗

v0.6 · 含 Gemini AI 整本生成

✨ 它幫你做什麼？

  • 用 AI 一鍵生整本繪本（故事 + 插圖 + 題目）
  • 變成可掃 QR Code 的 Google 表單測驗
  • 學生答完，老師自動拿到回應
  • 免裝程式，瀏覽器就能跑（手機/平板/電腦都行）
1

拿一支免費 Gemini API Key

⏱ 30 秒

到 Google AI Studio（用學校或個人 Google 帳號登入）→ 點「Get API key」→「Create API key」→ 複製那串 AIzaSy...。

📋 複製整串 API Key 備用
2

把 Key 貼進工作坊

⏱ 30 秒

打開 cagoooo.github.io/storytell。頂部展開 🤖 Gemini AI 助手。

  1. 把剛才複製的 API Key 貼到 API Key 欄位
  2. 按 💾 儲存（Key 只在你的瀏覽器，不會上傳）
  3. 按 🧪 測試連線，看到「✓ 連線正常」就行
3

一鍵生整本繪本（含插圖）

⏱ 約 90 秒

捲到「步驟 5」展開 🪄 用 Gemini 一鍵生成整本繪本。

  1. 主題填例如「勇氣」「友誼」「環保」
  2. 選年級（國小 1-6）、章節數（5 章剛好）
  3. 勾選 🖼 順便為每章生圖 ← 用 AI 自動配插圖
  4. 按 ✨ 開始生成
⏳ 生故事 (10 秒) → 生圖 (5×15 秒) → 完成。每章故事 + 圖 + 題目都會自動填好。
4

產生 Google 表單程式碼

⏱ 30 秒

編輯器內檢查每章內容（不滿意可按 ✨ 簡化故事 / ✨ 加長故事 / ✨ 換個寫法 微調）。

  1. 按 ⚡ 一鍵產生 Apps Script
  2. 跳出黑色程式碼框 → 按 📋 複製程式碼
5

跑 Apps Script 變表單

⏱ 60 秒
  1. 按 App 內 🚀 開啟 Apps Script（或到 script.google.com）
  2. 「新增專案」→ 把剛才複製的程式碼貼上 → 按 儲存（💾 圖示）
  3. 選擇要執行的函式 buildForm → 按 執行
  4. 第一次跳授權：選你的 Google 帳號 → 「進階」→「前往（不安全）」→ 允許
  5. 下方「執行紀錄」會印出 填表網址 ← 這就是學生用的表單連結！
6

印 QR Code 派發給學生

⏱ 30 秒
  1. 回到 App 步驟 6，把上面拿到的填表網址貼進去
  2. 填班級 / 老師姓名（會印在派發單上）
  3. 按 🖨 列印 A4 派發單 → 印出來貼班級公布欄 / 發給學生
  4. 學生用手機掃 QR → 跟著故事答題 → 完成提交
🎉 老師可在 Google 表單的「回應」分頁查看每個學生的作答

💡 加分技巧（看到這裡你已經會用了）

  • 多本繪本：頂部 📚 下拉切換 — 每本獨立進度，互不影響
  • 7 種題型：步驟 5 上方下拉「🎯 題型範本」可換成是非題、閱讀理解、看圖說故事...
  • 離線可用：footer 按 📲 安裝到桌面 → 變獨立 App，斷網也能編輯
  • iPad 加入主畫面：Safari 開網址 → 分享 ⬆ → 加入主畫面
  • 深色模式：footer 點 🌙 → 晚上備課護眼
  • 不想用 AI 生圖：自己用 Gemini 圖文生繪本 → 貼 Google Doc → 步驟 3 抓圖
  • 免費額度：gemini-2.5-flash 每天 1500 次請求，老師日常用永遠免費
